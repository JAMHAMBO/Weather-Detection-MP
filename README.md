# Weather App

A simple Weather App built using HTML, CSS, and JavaScript that fetches real-time weather data using the OpenWeatherMap API.

## Features

- Search weather by city name
- Displays current temperature
- Displays city name
- Shows humidity percentage
- Shows wind speed
- Dynamic weather icons based on weather conditions
- Input validation for empty searches
- Error handling for invalid city names
- Responsive and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API
- OpenWeatherMap API

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/JAMHAMBO/Weather-Detection-MP.git
   ```

2. Open `index.html` in your browser.

3. Generate a free API key from OpenWeatherMap and replace it in `script.js`:

   ```javascript
   const apiKey = "YOUR_API_KEY";
   ```
